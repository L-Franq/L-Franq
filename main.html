Projectos

Adelidocinhos

Um Web App de gestão de uma pastelaria unipessoal, desenvolvido utilizando (HTML, CSS e JS) vanilla para o frontend, Node.js para o backend e PostgreSQL como banco de dados. O sistema inclui funcionalidades como cadastro de usuários, marcação por calendário, processamento de pedidos e geração de relatórios de vendas.

RaedaMotors

A demo (caracter escolar) de um Web App de gestão de uma de uma stand de automóveis, desenvolvido utilizando (HTML, CSS e JS) vanilla para o frontend, PHP para o backend. O sistema inclui funcionalidades como cadastro de usuários, marcação, listagem, processamento de pedidos e geração de relatórios de vendas.

Temperos da Tchocha

O outro lado

Além do desenvolvimento de software, tenho uma paixão por explorar diferentes idiomas a volta do globo, o que me levou a aprender sobre variadas culturas.
